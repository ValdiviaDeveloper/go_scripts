# Código de Conducta

## Propósito

Este proyecto está comprometido con proporcionar un espacio seguro, respetuoso e inclusivo para todos los contribuyentes y usuarios.

## Estándares

Se espera que todos los participantes:

- Usen este proyecto **solo para propósitos autorizados y legales**
- Respeten la privacidad y seguridad de sistemas ajenos
- Cumplan con todas las leyes aplicables
- Traten a otros participantes con respeto
- No hagan spam, hagan mal uso, o ataquen a otros

## Escaneos responsables

- ✅ Solo autoriza escaneos en sistemas que controles
- ✅ Obtén consentimiento escrito cuando sea requerido
- ✅ Respeta límites de recurso del sistema objetivo
- ✅ No hagas DoS (denegación de servicio)

## Comportamiento inaceptable

- Escanear sistemas sin autorización
- Intentar explotar vulnerabilidades
- Acoso o discriminación
- Spam o contenido inapropiado
- Violación de privacidad ajena

## Reporte

Si vas a reportar un problema, incluye:
1. Descripción clara del problema
2. Versión del software
3. Sistema operativo usado
4. Pasos para reproducir (si aplica)

## Responsabilidad legal

El uso de este proyecto debe cumplir con todas las leyes aplicables. No somos responsables por mal uso.

Gracias por ser parte de esta comunidad responsable.
